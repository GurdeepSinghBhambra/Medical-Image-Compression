GUIDELINES

1) Create a folder having only PNG, Gray Scale images of chest.
2) For encoding, the images should be of 1024 x 1024 dimension.
3) For decoding, images encoded using this website should be used.
4) All the images should be in a zip folder with no sub folders.
5) File size should not exceed 3 MB in size.
