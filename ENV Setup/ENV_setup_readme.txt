Environment Setup:
1. Install Anaconda from: "https://www.anaconda.com/". Install 64bit, windows version
2. Open "anaconda prompt" using search bar in taskbar.
3. Now navigate to the folder where "ENV_requirements.txt" file is present (from the terminal XD).
4. Enter the command: "conda create --name minor --file ENV_requirements.txt" and you are done.
5. activate environment by entering "conda activate minor" or to deactivate use "conda deactivate minor".
